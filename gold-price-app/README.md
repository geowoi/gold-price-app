# Gold Price Prediction App

Aplikasi kecerdasan buatan berbasis Python untuk analisis statistik dan prediksi harga emas.

## Fitur
- Grafik harga emas historis
- Statistik harga emas
- Prediksi harga emas menggunakan Machine Learning
- Aplikasi web publik

## Teknologi
- Python
- Streamlit
- Scikit-Learn
- Yahoo Finance Dataset
